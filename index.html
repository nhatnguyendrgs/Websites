Chèn danh sách

Để chèn một danh sách ta dùng element ol (viết tắt của odered list) và li, ví dụ:

  1. Facebook
  2. Google
  3. Yahoo
  4. Git Hub

Ngoài ra có thể dùng element ul (viết tắt của unordered list) nếu không đếm thứ tự, ví dụ:

  • Facebook
  • Google
  • Yahoo
  • Git Hub

5. Tạo bảng

Để tạo bảng ta cần sử dụng các element riêng biệt, dưới đây là một bảng ví dụ:

Những element tạo bảng
Element Tác dụng
table Giống như các element ol, ul và dl bao gồm toàn bộ danh sách của nó.
caption Element đưa ra tiêu đề cho bảng
tr Element đại diện cho một hàng của bảng
th Element miêu tả các ô tiêu đề
td Element miêu tả các ô bình thường
